# 配信向け・ドラマ主役UI

GitHub上の以下2ファイルを丸ごと差し替えてください。

- `index.html`
- `style.css`

`main.js` は変更不要です。

## 変更内容

- 配信用なので操作ボタンを画面から非表示
- JavaScriptが参照する要素は隠して残しているため動作は維持
- ドラマ欄を画面中央の主役として大型化
- 吹き出し表現を強調
- マップは上段の状況説明
- 年代記は右側の補助情報
- 世界・文明・キャラ数値は下段へ圧縮
- 1920×1080配信向け

## OBS設定例

- 幅: 1920
- 高さ: 1080
- FPS: 30
